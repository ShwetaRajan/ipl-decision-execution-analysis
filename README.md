# When Data Challenges Intuition: Decision and Execution Patterns in IPL

## Overview
This project presents a data-driven analysis of decision-making and execution patterns in IPL matches using ball-by-ball data across multiple seasons.

The analysis evaluates whether commonly followed strategies, such as toss decisions, align with actual match outcomes. The findings reveal that toss decisions result in nearly identical win rates, challenging conventional assumptions.

## Key Insight
Match outcomes are not driven by the decisions teams believe matter, but by how effectively they execute when it matters most.

## Approach
The analysis is structured across multiple layers:
- Decision Layer (toss behavior)
- Outcome Layer (win rates)
- Execution Layer (phase-wise scoring)
- Interaction Layer (phase × toss decision)
- Context Layer (venue)

## Files
- PDF report: Final analysis and insights
- Power BI file: Interactive dashboard

## Dataset
Dataset sourced via Kaggle (originally derived from Cricsheet).
Ball-by-ball data covering IPL seasons 2008–2025.
